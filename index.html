🔗 Conectare GitHub

Datele tale vor fi salvate intr-un Gist privat pe contul tau GitHub si sincronizate automat intre toate dispozitivele.

Pas 1 — Creaza un Personal Access Token:
Mergi la github.com/settings/tokens/new
• Note: SubTracker
• Expiration: No expiration
• Bifeaza ✓ gist
• Click Generate token → copiaza tokenul

Pas 2 — Lipeste tokenul mai jos si apasa Conecteaza.
Aplicatia va crea automat un Gist privat pentru datele tale.

🔒 Tokenul este salvat doar local pe acest dispozitiv (localStorage). Nu este transmis nicaieri altundeva decat catre api.github.com.
